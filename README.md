# Smart Number Guess Challenge

A Java Swing based desktop application that generates a random number and challenges the user to guess it. The application provides hints, tracks attempts, and offers an interactive GUI for a better user experience.

## Features

- User-friendly Java Swing GUI
- Random number generation (1-100)
- Hint system (Too High / Too Low)
- Maximum 10 attempts
- Best score tracking
- Progress bar
- Restart game option
- Exit button
- Input validation
- Error handling using dialog boxes

## Technologies Used

- Java
- Java Swing
- OOP
- Event Handling

## How to Run

Compile:

```bash
javac SmartGuessGame.java
